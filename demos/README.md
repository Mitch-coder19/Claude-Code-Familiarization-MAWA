# 📂 Demos by Function

Each folder is self-contained: a `README.md` with copy-paste prompts plus the synthetic sample files those prompts use. `cd` into your function's folder, launch `claude`, and paste a prompt.

> ⚠️ **Synthetic data only.** Every sample file here is 100% fabricated. Never substitute real PHI, donor, or wish-kid data.

| 📊 **Business Intelligence** | [`bi/`](bi/) | SQL, DAX, pipelines, documenting the undocumented. |
| 🏗️ **InfraArch** | [`infraarch/`](infraarch/) | Infrastructure-as-Code, Azure governance, AI prototyping. |
