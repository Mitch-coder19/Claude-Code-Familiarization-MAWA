# 🕐 60-Minute Run of Show

**Session:** Agentic Coding with Claude Code — Familiarization
**Audience:** Make-A-Wish America IT Enablement Cohort (5 functions, 18 people)

| Time | Segment | What happens |
|---|---|---|
| **0–10** | 🎬 **Opener + ground rule** | Welcome. What agentic coding is (not autocomplete — reads files, runs commands, iterates). Set the **⚠️ Synthetic Data Only** ground rule up front: no real PHI, donor, or wish-kid data in prompts. Frame the goal: delegate multi-step technical work. |
| **10–20** | ⭐ **Flagship live demo** | Lead with **Product Management's PRD → clickable prototype** (`demos/product`, Demo 1) — highest wow, lands hardest with the "I don't code" crowd. Show the before (blank slate) → after (working HTML mockup) in one prompt. |
| **20–45** | 🔄 **Rotating 5-min demos** | One flagship per function, ~5 min each: **BI** optimize the slow query (`demos/bi`) · **InfraArch** audit `main.bicep` for misconfigs (`demos/infraarch`) · **Engineering** runbook → PowerShell script (`demos/engineering`) · **Governance** PII/PHI scanner on `patients_sample.csv` (`demos/governance`). Keep each to one before→after. |
| **45–55** | 🙌 **Hands-on** | Everyone runs **one prompt from their own function folder**. Facilitators float and help. `cd demos/<function>` → `claude` → paste the flagship prompt. |
| **55–60** | 🏁 **Close** | How to get access to Claude Code, where to get help, and the bring-your-own-task follow-up: each person names **one real task from their week** to try next. |
